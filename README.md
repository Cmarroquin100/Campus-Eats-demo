# Campus Eats

This application  is dessigned to provide students of Maryville University the opportunity to collect leftover food from school campus events designed with a timeframe and admin access to post food and a disclaimer for allergens that may be contained in the food. 

A **mobile app prototype** Built with Expo (React Native) for an 8-week app design class.

## Features (prototype)

- **Home Feed** – Filter tabs (All, Vegetarian, DUC, Gander Hall) and food cards with location, timer, tags, “Low Quantity”
- **Post Leftovers (Staff only)** – Approved staff/organizers can post: event name, location (fixed list + custom), room, description, photo placeholder, timer (15 / 30 / 60 min), “Notify Students”
